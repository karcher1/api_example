# Archive Manifest

This archive contains documentation requirements and starter content for a file-driven API documentation website.

- `README_ADD_TO_PROJECT.md` — Russian instructions for adding the archive to a project.
- `docs/API_DOCS_PLATFORM_SPEC.md` — complete functional specification.
- `docs/API_DOCS_YAML_SCHEMA.md` — YAML/content schema.
- `docs/CODEX_IMPLEMENTATION_PROMPT.md` — ready-to-use prompt for Codex.
- `docs/CODEX_TASK_CHECKLIST.md` — acceptance checklist.
- `docs/CONTENT_MANAGEMENT_GUIDE.md` — guide for adding/editing endpoints and articles.
- `content/api/navigation.yaml` — example API navigation.
- `content/api/endpoints/create-user.yaml` — example endpoint.
- `content/api/endpoints/get-user.yaml` — example endpoint.
